## Предустановка

`git clone git@github.com:UMBARLOCE/ncs-color.git`

`cd ncs-color`

`python3 -m venv venv`

`source venv/bin/activate`

`pip install --upgrade pip`

`pip install -r requirements.txt`


## Установка

one time; создание папки с картинками

`python generate_jpg.py`

## Запуск

`python main.py`
